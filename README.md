# SmartScrape
A Python based web scraping project that extracts book data from Books to Scrape using Requests and BeautifulSoup.

## Features
- Extracts:
  - Book titles
  - Prices
- Uses:
  - Python
  - Requests
  - BeautifulSoup4
- Beginner-friendly scraping workflow
- Git and GitHub integrated

## Website Used
https://books.toscrape.com/

## Installation

```bash
pip install -r requirements.txt
```

## Run Project
python app.py

## Technologies Used
- Python
- BeautifulSoup4
- Requests
- Git/GitHub
